HTML stands for High Text Markup Language. It is the core language of the world web.HTML elements tell the browser how to display the content. It is used to define the structure and layout of web documents by using various tags.It can be assisted by technologies such as Cascading Style Sheet(CSS)and scripting languages such as Javascript. Although a fundamental language of the web,HTML is a static language content created with it does not change. HTML is used to specify the way webpages look,not how they function.
